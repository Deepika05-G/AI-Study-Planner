# AI-Based Personalized Study Planner

This project is designed to help students create personalized study schedules based on their subjects, available time, and priorities.

## Features
- Takes user input for subjects and available study hours
- Generates a structured study timetable
- Helps improve productivity and time management

## Technologies Used
- Python
- HTML
- CSS

## Purpose
This project was built as a mini project to apply logical thinking and problem-solving skills to a real-world student problem.
